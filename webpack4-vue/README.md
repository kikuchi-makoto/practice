# webpack4-vue

## Vue.js開発環境

- webpack ^4.29.5
- webpack-dev-server
- babel
- style(MiniCssExtractPlugin.loader)/css/postcss/sass
- extensions .js/.vue
- Chunks(all)
- minify to style.css
- eslint for Vue.js
- sideEffects: false,
